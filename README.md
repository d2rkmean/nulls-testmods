# Null's TestMods
---
An Android utility for installing and testing Null's Brawl mods and more!

### Features
- Installing unsigned .zip/.json mods (for testing)
- Viewing mod information
- Viewing mods from the Library
- Adding your own library repositories
- Viewing signature information
- Much more...

### Build
```bash
git clone https://github.com/d2rkmean/nulls-testmods.git
cd nulls-testmods
./gradlew assembleDebug
```
